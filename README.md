# Description
An offline python implementation of the blockposters website

# How it works
input an image

default is to print in A4 page sizes

will select how many pages it should be, or maybe select number of pages for width

image gets resized accordingly

image is padded on right and bottom sides to accomodate slicing

image is sliced into tiles

tiles are named numerically

tiles are saved to a temporary folder

tiles are recompiled into a pdf file

temporary folder is deleted
